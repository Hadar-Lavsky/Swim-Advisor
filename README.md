
### Troubleshooting

If you see "'react-scripts' is not recognized" error on Windows:
1. Make sure you've installed react-scripts globally
2. Try cleaning and reinstalling:
```bash
rm -r node_modules
npm cache clean --force
npm install
npm install -g react-scripts
```
